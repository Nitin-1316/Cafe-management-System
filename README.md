# Cafe Management System

A complete SQL + Python based system to manage cafe operations like billing, inventory, and sales tracking.

Features:
- Menu Management:* Add, update, delete food items with prices
- Billing System:* Auto-generate bills with GST calculation  
- Inventory Tracking:* Track stock of ingredients and alert on low stock
- Sales Reports:* Daily/Monthly sales analysis using SQL queries
- Customer Records:* Store customer details for loyalty program

 Tech Stack:
- Database: MySQL
- Backend: Python
- Queries Used: JOINs, Subqueries, Aggregate Functions, Stored Procedures

Database Schema:
Tables: menu, orders, order_items, customers, inventory

 How to Run:
1. Import cafe_management.sql into MySQL
2. Run python main.py for billing interface
